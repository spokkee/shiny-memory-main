Question 31 : Earliest expected time of completion for an activity is found using

  1. Expected time calculation
  2. Forward Pass method
  3. Backword Pass Method
  4. Crashing
  
Forward Pass method

Question 32 : Latest starting date for an activity is estimated using

  1. Resource leveling
  2. Backword Pass Method
  3. Forward Pass method
  4. Crashing
  
Backword Pass Method

Question 33 : Select the correct statement from the following

  1. Slack or float of dummy activity is always equal to zero
  2. There is always only one critical path in the network
  3. A path is called a critical path if it is the longest path in a project network
  4. Crashing cost linearly increases with no of days crashed
  
A path is called a critical path if it is the longest path in a project network

Question 34 : Why does activities on critical path of a CPM network are called critical

  1. They represent maximum project completion time
  2. They cannot tolerate any delay in completion
  3. They consume maximum rresources
  4. These are most cpmplex activities on project
  
They cannot tolerate any delay in completion

Question 35 : If an activity with free slack time of 2 weeks is delayed by 1 week

  1. the project will be delayed by 1 week.
  2. the slack time of all activities that follow this activity is reduced by 1 week.
  3. no other activity in the project is affected.
  4. the probability of completing the project on time decreases.
  
no other activity in the project is affected.

Question 36 : When many activities are planned to start at the same time in project schedule, the project is likely to be following

  1. Goldratt's Critical Chain
  2. Concurrent Engineering
  3. Research and Development Project
  4. laddering approach
  
Concurrent Engineering

Question 37 : A projeect has three independent critical paths A (Having maximum no. of activities), B(Using maximum rsources) and C(Having maximum crashing cost). To reduce the project length, we have to crash

  1. Activities on Path A
  2. Activities on Path B
