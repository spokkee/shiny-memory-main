Question 1 : Difference between star and snowflake schema is______

  1. One fact table and many dimension tables
  2. Many fact table and many dimension tables
  3. Normalizing dimension tables
  4. Functional Dependency
    
Normalizing dimension tables

Question 2 : A cross-functional business process that originates when the company receives a customer order and concludes when it receives a payment from the customer is _____

  1. Order fulfillment process
  2. Control Process
  3. Manufacturing process
  4. Procurement process
    
Order fulfillment process

Question 3 : Pervasive computing is also known as ________

  1. Super computing
  2. Ubiquitous computing
  3. Mainframe computing
  4. Satellite computing
    
Ubiquitous computing

Question 4 : What is Unethical is not necessarily_____

  1. Legal
  2. Illegal
  3. Secure
  4. Unsecure
    
Illegal

Question 5 : A cyber attack in which an attacker sends a flood of data packets to the target computer, with the aim of overloading its resources is refered to as__________

  1. Spyware
  2. Phishing attack
  3. Cyberwarfare
  4. Denial of service
    
Denial of service

Question 6 : The information system which collects data continuously, typically in real time is_________

  1. Business Intelligence System
  2. Expert System
  3. Transaction Processing System
  4. Database Systems
    
Transaction Processing System

Question 7 : A relational database consists of a collection of _____________

  1. Tables
  2. Fields
  3. Records
  4. Keys
    
Tables

Question 8 : Backbone of any organisation is____________

  1. Employee
  2. Information
  3. Management
  4. Capital
    
Information

Question 9 : The data in the datawarehouse is described by:

  1. Relational data
  2. Operational data
  3. Informational data
  4. Meta data
    
Meta data

Question 10 : Business processes NOT supported by ERP systems is ______

  1. Daily Transactions
  2. Procurement process
  3. Fulfillment process
  4. Production process
    
Daily Transactions

Question 11 : Which web technique allows viewers to write short messages

  1. Email
  2. Review Sheet
  3. Forum
  4. Microblogging
    
Microblogging

Question 12 : Which is NOT a fundamental tenet of ethics-

  1. Privacy
  2. Responsibility
  3. Accountability
  4. Liability
    
Privacy

Question 13 : A datawarehouse helps buisness executives to

  1. Archive the important Data
  2. use their data for decision making
  3. organize data
  4. create real time transactions
    
use their data for decision making

Question 14 : Organizations usually are not using social computing in which of the following functions?

  1. Marketing
  2. Customer Relationship Management
  3. Human Resource Management
  4. Production
    
Production

Question 15 : Which is not a computer based information system

  1. Customer relationship management
  2. Supply chain management
  3. Visionary management
  4. Operation management
    
Visionary management

Question 16 : The purpose of data warehouse is to provide________data that is not stored in the operational environment.

  1. Summary
  2. Normalized
  3. Informational
  4. Denormalized
    
Summary

Question 17 : The department responsible for payroll preparation and other various benefits is ______

  1. Customer relations Management
  2. Human Resources Planning and Management
  3. Accounting and Finance
  4. Financial planning
    
Human Resources Planning and Management

Question 18 : Businesses have become dependent on computer networks for improving their_________

  1. Efficiency
  2. Accuracy
  3. Interactions
  4. Information exchange
    
Efficiency

Question 19 : An instruction in a computer program that triggers a malicious act is called as _________

  1. Logic Bombs
  2. Packet Sniffer
  3. Hacking
  4. Computer Crime
    
Logic Bombs

Question 20 : _______________is the practice of marketing additional related products to customers based on previous purchases.

  1. Up-selling
  2. Cross-Selling
  3. Down-selling
  4. Seasonal-Selling
    
Up-selling

Question 21 : Extended service set in a wireless network comprises of __________

  1. All access points
  2. All network points
  3. All connected access points
  4. Connected basic service sets
    
Connected basic service sets

Question 22 : In ____________, needed services such as temporary help or extra space are traded on an as-needed basis.

  1. Horizontal exchanges
  2. Vertical exchanges
  3. Personal exchanges
  4. Functional exchanges
    
Functional exchanges

Question 23 : _____ reports show a greater level of detail than is included in routine reports

  1. Drill-down
  2. Key-indicator
  3. Routine
  4. Ad hoc
    
Drill-down

Question 24 : To effectively manage reports by exception, the company must first create:

  1. Performance standards
  2. Best practices
  3. User information requirements
  4. A database
    
Performance standards

Question 25 : _____ is a CRM system that is hosted by an external vendor in the vendor's data center.

  1. Analytical CRM
  2. On-demand CRM
  3. Operational CRM
  4. Customer-facing CRM
    
On-demand CRM
